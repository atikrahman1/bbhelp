Manage Exclusion List

Excluded domains will NOT be auto-scanned.

Enter one domain pattern per line. Supports wildcards:

  • *.google.com - Excludes all Google subdomains (mail.google.com, drive.google.com, etc.)
  • example.com - Excludes only example.com
  • *.example.* - Excludes all subdomains and TLDs
Example:
*.google.com
localhost
127.0.0.1
*.internal.company.com
