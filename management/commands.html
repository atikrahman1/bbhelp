Available Variables:
  • {URL} - Full URL with path and parameters (e.g., https://test.com/path?param=value)
  • {DOMAIN} - Domain without www (e.g., test.com)
  • {ORIGIN} - Protocol + domain (e.g., https://test.com)
  • {TARGET} - Domain with port if any (e.g., test.com:8080)
  • {HOST} - Same as TARGET
  • {PROTOCOL} - http or https
  • {PATH} - URL path (e.g., /api/users)
Example:
Name: SQLMap Scan
Command: python3 sqlmap.py -u {URL} --dbs
