Manage Google Dorks

Available Variable:
  • {DOMAIN} - Current domain without www (e.g., test.com)
Common Google Dork Operators:
  • site: - Search within a specific site (e.g., site:{DOMAIN})
  • inurl: - Search for URL containing keyword (e.g., inurl:admin)
  • filetype: - Search for specific file types (e.g., filetype:pdf)
  • intitle: - Search in page title (e.g., intitle:"index of")
  • intext: - Search in page content (e.g., intext:password)
  • ext: - File extension (e.g., ext:sql)
Examples:
Name: Database Backups
Dork: site:{DOMAIN} ext:sql | ext:db | ext:dump

Name: Directory Listings
Dork: site:{DOMAIN} intitle:"index of" inurl:backup

Name: Exposed Credentials
Dork: site:{DOMAIN} intext:"password" filetype:txt
