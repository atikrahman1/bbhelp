Manage Tools

Available Variables:
  • {URL} - Full URL with path and parameters
  • {DOMAIN} - Domain without www (e.g., test.com)
  • {ORIGIN} - Protocol + domain (e.g., https://test.com)
  • {TARGET} - Domain with port if any
Examples:
Name: SecurityTrails
URL: https://securitytrails.com/domain/{DOMAIN}/dns

Name: VirusTotal
URL: https://www.virustotal.com/gui/domain/{DOMAIN}

Name: URLScan
URL: https://urlscan.io/search/#{DOMAIN}

Name: Wayback Machine
URL: https://web.archive.org/web/*/{DOMAIN}
